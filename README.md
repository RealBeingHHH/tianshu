# 天枢 Tianshu

> 信任不需要担保人。信任只需要不可篡改的记录。

---

## 这是什么

天枢是一个**物理层不可篡改的验证基础设施**。

你在用 AI 代理帮你干活。你怎么知道它真的干了？你怎么知道验证它的那个东西没有被改过？你怎么知道帮你备份的系统没有帮倒忙？

天枢回答的是最后一个问题——**那个看守一切的东西，谁来保证它没有被改过？**

答案：物理定律。你改它，它就坏。不是"它报告自己被改了"——是它**不能在被改之后继续工作**。

---

## 不是什么

- ❌ 不是"更好的 AI"——天枢不判断对错，不分析内容
- ❌ 不是密码学产品——加密可以被密钥攻破，物理破坏不行
- ❌ 不是商业软件——不推销，不需要你"信任"，需要你**验证**
- ❌ 不是一键安装的 SaaS——你需要把它部署到你自己的硬件上
